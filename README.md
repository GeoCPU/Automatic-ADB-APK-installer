# Automatic ADB APK Installer

A professional tool to quickly and automatically install APK files on Android devices via ADB.

---

## Requirements

1. **ADB (Android Debug Bridge)**  
   Install the **ADB installer**.
   Follow all the instructions with Y for each step. 

3. **Python 3.x**  
   Download and install Python from [python.org](https://www.python.org/downloads/).  
   During installation, make sure to check **“Add Python to PATH”**.

---

## Installation & Usage

1. **Download the script**  
   Save `adbapkinstaller.py` to a folder on your computer.

2. **Connect your Android device**  
   - Enable **Developer Options → USB Debugging** on your device.  
   - Connect your device via USB.

3. **Launch the program**  
   Open a terminal or command prompt, navigate to the folder containing the script, and run:

   ```bash
   python adbapkinstaller.py

4. **Finish.**
 Put in your APKs, select your device, and install them to your device! This completes the installation process using the graphical interface.
